# Summary 
This project is a responsive Media application that allows users to browse and interact with media content through a clean, fast, and modern UI.

# Technology Used
  Front-end
- HTML5
- CSS3
- JS
- Bootstrab

# Brief About the project
- This project is a modern and responsive Media application that allows users to browse, search, and view different types of media content smoothly.
- It focuses on delivering a clean and intuitive UI/UX with fast performance and organized front-end structure.
- The app is built using HTML, CSS, JavaScript, and local JSON data to provide a seamless and dynamic media browsing experience.
